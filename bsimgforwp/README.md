# BSImg 图床 for WordPress

将 WordPress 媒体库上传的图片接管到 [BSImg 图床（bsimgbed）](https://github.com/ilolioo/bsimgbed)，减轻站点存储压力并统一图片管理。

## 功能

- 在 WordPress 后台上传图片时，自动将文件上传到 bsimgbed 图床
- 前台显示的图片地址使用图床 URL，不占用主机存储
- 支持**公共上传**（无需 API Key）和**私有上传**（需在图床后台配置 API Key）
- 可选**储存桶 ID**，配合图床多储存桶使用
- 保存设置时保留已填写的 API Key（密码框留空不覆盖）
- 设置页保存成功后显示提示
- **从 URL 导入**的图片也会同步到图床
- REST API（古腾堡编辑器、移动端等）返回的附件地址为图床 URL
- 设置页提供**测试图床连接**，一键验证配置
- 设置页采用卡片式布局与统一样式，更清晰易用
- **高级选项**：可配置上传请求超时、是否验证图床 SSL 证书

## 环境要求

- WordPress 5.0+
- PHP 7.4+
- 已部署并可访问的 bsimgbed 图床服务

## 安装

1. 将 `bsimgforwp` 文件夹上传到 WordPress 的 `wp-content/plugins/` 目录
2. 在 WordPress 后台 **插件** 中启用 **BSImg 图床 for WordPress**
3. 打开 **设置 → BSImg 图床**，填写图床地址并选择上传方式

## 配置

| 选项       | 说明 |
|------------|------|
| **图床地址**   | 您的 bsimgbed 站点根地址，例如 `https://img.example.com`，不要末尾斜杠 |
| **上传方式**   | **公共上传**：使用图床的 `/api/upload/public`，需在 bsimgbed 后台开启“公共上传”<br>**私有上传**：使用 `/api/upload/private`，需填写 API Key |
| **API Key**    | 在 bsimgbed 后台 **API 管理** 中创建，仅私有上传时必填。留空则保留已保存的 Key |
| **储存桶 ID**  | 可选。图床多储存桶时填写，留空使用图床默认储存桶 |
| **请求超时**   | 高级。上传请求超时时间 5–120 秒，默认 30 |
| **SSL 证书验证** | 高级。建议开启；自签名证书图床可关闭 |

## 使用说明

- **新上传的图片**：在媒体库中上传后，会自动同步到图床，文章/页面中插入的图片地址为图床链接
- **已有附件**：本插件不会自动把历史附件迁移到图床；如需迁移，可重新上传或自行替换 URL
- 图床需允许您的 WordPress 站点所在服务器访问上传接口；若图床有 IP 限制，请将服务器 IP 加入白名单

## 与 bsimgbed 的对应关系

- 公共上传：`POST {图床地址}/api/upload/public`，表单字段 `file`
- 私有上传：`POST {图床地址}/api/upload/private`，请求头 `X-API-Key: 您的Key`，表单字段 `file`
- 可选表单字段 `bucketId`（在设置中填写储存桶 ID 时发送）
- 上传成功后，图片访问地址为：`{图床地址}/i/{uuid}.{格式}`

## 更新日志

### 1.4.0

- 设置页新增**测试图床连接**按钮，上传一张最小测试图以验证配置
